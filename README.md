# Интеграционный проект: API + PostgreSQL

## Цель проекта
Создать работающую интеграцию REST API с базой данных PostgreSQL для демонстрации навыков интеграционного тестирования.

## Что сделано
1. **API сервер** на Node.js + Express с 4 endpoints:
   - `GET /posts` - получение всех постов
   - `GET /posts/:id` - получение конкретного поста
   - `POST /posts` - создание нового поста
   - `DELETE /posts/:id` - удаление поста

2. **База данных** PostgreSQL с таблицей `posts`

3. **Полная интеграция**: данные, созданные через API, сохраняются в БД

## Технологии
- Node.js + Express
- PostgreSQL
- Postman (тестирование API)
- DBeaver (работа с БД)

## Как запустить
1. Установи PostgreSQL и создай базу данных
2. Установи Node.js
3. Склонируй репозиторий
4. Установи зависимости: `npm install`
5. Настрой подключение к БД в `server.js`
6. Запусти сервер: `node server.js`
